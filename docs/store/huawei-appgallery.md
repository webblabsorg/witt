## Full Description

Witt is the AI-powered study companion built for every student, everywhere.

Prepare for SAT, GRE, WAEC, JAMB, IELTS, and 100+ standardized exams with adaptive AI, smart flashcards, mock tests, and a personal AI tutor available 24/7.

**Key Features:**
- Sage AI Tutor — step-by-step homework help and personalized study plans
- Test Prep Engine — adaptive questions for 100+ exams
- Smart Flashcards — 5 study modes with spaced repetition
- Mock Tests — full-length timed simulations
- Educational Games — 9 games + daily brain challenges
- Study Community — groups, forum, deck marketplace
- Full Offline Support — download content packs, works in airplane mode
- 21 Languages supported

**HMS Integration:**
- HMS Push Kit for push notifications
- HMS Account Kit for sign-in (Google Sign-In not available on Huawei devices)
- Huawei IAP via Subrail SDK for in-app purchases
